## 建表语句
deal_TB表
```mysql
mysql> create table deal_TB
    -> (
    -> goodsID smallint primary key,
    -> name varchar(10) not null,
    -> type smallint not null,
    -> price smallint not null,
    -> marketID smallint not null,
    -> producerID smallint not null,
    -> foreign key(type) references goodsType_TB(type),
    -> foreign key(marketID) references market_TB(marketID),
    -> foreign key(producerID) references producer_TB(producerID)
    -> )
    -> ;
Query OK, 0 rows affected (0.02 sec)
```
